Who We Are

Perfume World is a simple perfume website created to show different types of perfumes. The website is designed using HTML, CSS, and JavaScript with responsive design.

Our Goal

Our goal is to help users explore perfumes in an easy and attractive way. We want the website to be simple, clear, and suitable for all screen sizes.
